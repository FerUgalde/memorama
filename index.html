Las declaraciones if y else son estructuras de control fundamentales en la programación que permiten tomar decisiones en el código basadas en ciertas condiciones.
La declaración if se utiliza para ejecutar un bloque de código si una condición específica es verdadera.
La declaración else se utiliza junto con if para ejecutar un bloque de código si la condición en la declaración if es falsa


if (condicion) {
// Bloque de código que se ejecuta si la condición es verdadera
} else {
// Bloque de código que se ejecuta si la condición es falsa
}
